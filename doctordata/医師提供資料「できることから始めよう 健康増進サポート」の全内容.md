# できることから始めよう 健康増進サポート

## 「主食」「主菜」「副菜」の働きと1食の目安（Page 1）

### 主食（炭水化物を多く含む食材：ご飯、パンなど）

* 
**働き**: 脳や神経組織、赤血球などのエネルギー源になり、筋肉を動かす上でも必要です 。


* 
**不足すると**: 集中力低下や倦怠感など体調不良の原因になります 。


* 
**摂りすぎると**: 体重増加につながり、生活習慣病のリスクが高くなります 。


* **過剰摂取で悪化する数値**:
* 血圧（原因：主菜の塩分）


* 血糖値（原因：炭水化物）


* 中性脂肪（原因：炭水化物）




* 
**1食あたりの目安**: 「握りこぶし」の量 。



#### 炭水化物を摂る際のポイント

* 
**副菜から食べる**: 食物繊維→たんぱく質→炭水化物の順で食べると、吸収が穏やかになり食後血糖の急激な上昇が抑制されます 。


* 
**精製度の低い物を選ぶ**: 雑穀米などは食物繊維が多いため、食後血糖値が上がりにくいです 。


* 
**遅い時間帯は量を減らす**: 代謝・活動量が低くなる寝る直前などは脂肪になりやすいです 。


* 
**他のおかずとの調整**: 主菜や副菜に脂質（揚げ物など）や糖質（イモ類など）が多い場合は主食を減らします 。

## 主菜の摂り方（Page 2）

### 主菜（たんぱく質を多く含む食材：魚、肉、大豆製品など）

* 
**働き**: 骨格、筋肉、皮膚、血液を作るもとになり、酵素やホルモンとして代謝を調節します 。


* 
**不足すると**: 筋肉量減少による代謝低下、免疫力の低下、貧血の原因になります 。


* 
**摂りすぎると**: 体重増加や生活習慣病リスク、尿酸上昇、腎機能低下につながります 。


* 
**1食あたりの目安**: 「指を含まない、片手のひら程度」の量（大豆製品の場合はこの2倍）。



#### たんぱく質を摂る際のポイント

* 
**3食ごとに適量摂る**: まとめ食べはできません。不足は筋肉減少、摂りすぎは脂肪増加になります 。


* 
**脂質の少ない部位を選ぶ**: 豚バラ（395kcal/100g）より豚ロース（263kcal/100g）を選ぶなど 。


* 
**魚や大豆製品を組み合わせる**: 肉の飽和脂肪酸は血栓リスクを上げますが、魚の脂や大豆のポリフェノールはリスクを下げます 。



カロリーカットの工夫 

* **脂身や皮を除く**:
* ロース肉の脂身をとる：-61kcal 


* 鶏もも肉の皮をとる：-77kcal 




* 
**ひき肉料理に注意**: 脂身と一緒にミンチにされているため高カロリーです（メンチカツ2個：690kcal、ハンバーグ1個：548kcal）。

## 副菜の摂り方（Page 3）

### 副菜（ビタミン・ミネラル・食物繊維：野菜、海藻、きのこなど）

* 
**目標量**: 1日350g（1食120g程度）。数値が高めの方は400g以上を推奨 。


* 
**構成**: 1/3は緑黄色野菜、2/3は淡色野菜で摂取するのが理想です 。



血糖値スパイクの抑制 

* 
**ベジファースト**: 野菜から先に食べることで、食後血糖値の急激な上昇（血糖値スパイク）を抑え、満腹中枢を刺激して食べ過ぎを防ぎます 。



食物繊維の種類と働き 

* 
**水溶性食物繊維**: 食物の移動を遅らせ血糖値の上昇を抑制。コレステロールを吸着・排出 。


* 
**不溶性食物繊維**: 便通を促進し、大腸がんを予防。満腹感を得やすい 。



食物繊維を多く含む食品（1食あたりの量）

* おから（50g）：5.8g 


* ひじき（70g）：2.8g 


* ブロッコリー（60g）：2.6g 


* ほうれん草（70g）：2.0g

## 塩分・糖質・果物（Page 4）

ドレッシングの比較（大さじ1杯/15gあたり）

| マヨネーズ | 106kcal | 0.3g |
| シーザー | 62kcal | 0.5g |
| ノンオイル和風 | 8kcal | 0.8g |

減塩のコツ 

* 1日6g未満を目標にします 。


* 香辛料、香味野菜、果物の酸味を利用する 。


* 麺類の汁は残す（全部残せば2〜3gの減塩）。



果物の目安 

* 1日あたりの適量は「握りこぶし1個分」程度 。


* 糖質が多いため、代謝が下がる夕方以降は避けて日中に摂りましょう 。

## 時間栄養学と食事のポイント（Page 5）

欠食のデメリット 

* 空腹時間が長くなると、次の食事で「血糖値スパイク」が起きやすくなります 。


* 1日3食規則正しく食べることが、代謝を維持し脂肪蓄積を防ぐ鍵です 。



遅い時間の食事の工夫 

* 
**分食のすすめ**: 夕食が遅くなる場合は、夕方に炭水化物を摂り、遅い時間におかず（主菜・副菜）のみを摂るようにします 。


* 寝る直前の場合は、炭水化物を減らし、脂質を控えた消化の良いメニューを選びましょう 。

## 選び方でカロリーダウン（Page 6）

調理法とメニューの選択 

* 
**丼もの**: カツ丼（1082kcal）→ 親子丼（796kcal）→ 海鮮丼（477kcal）。


* 
**調理法**: チキンカツ（428kcal）→ から揚げ（357kcal）→ 焼き鳥（234kcal）→ 蒸し鶏（154kcal）。



飲み物とお菓子 

* 
**飲み物**: 無糖やカロリーゼロを選択。スポーツ飲料（500ml）でも125kcal、スティックシュガー約8本分の糖質が含まれる場合があります 。


* 
**お菓子**: 1日100kcal以内を目安にし、15時までに食べ終えるのが理想です 。

## 運動と生活習慣病予防（Page 7-8）

運動の効果 

* 1日8,000〜10,000歩のウォーキングで高血圧や糖尿病などの予防効果が認められています 。


* 
**食べたら動く**: 食後10分程度の軽い動作（散歩やスクワット）で血糖上昇を抑制できます 。



おすすめ筋トレ3選 

1. 
**スクワット**: 下半身の大きな筋肉を鍛え、効率よく代謝を上げます（10〜15回×3セット）。


2. 
**プランク**: 体幹を鍛えます（30秒〜2分キープ）。


3. 
**クランチ**: 腹筋を鍛えます（10〜15回×3セット）。



インターバル速歩 

* 「さっさか歩き」3分と「ゆっくり歩き」3分を交互に繰り返します。筋力と持久力を同時に向上させます 。

## METs（メッツ）を使用したエネルギー計算（Page 9）

**消費エネルギー(kcal) = METs × 時間(時) × 体重(kg) × 1.05** 

主な活動のMETs値 

* 普通歩行（平地）：3.0 METs
* 掃除機、台所の仕事：3.3 METs
* ラジオ体操第1：4.0 METs
* 階段を上がる（ゆっくり）：4.0 METs
* ジョギング：7.0 METs

## 免疫力アップと睡眠（Page 10-11）

免疫力を高める4つの基本 

1. 
**食**: 主食・主菜・副菜を揃え、発酵食品や食物繊維で腸内環境を整える 。


2. 
**動**: 疲れない程度の「ながら運動」を習慣にする 。


3. 
**眠**: 寝る前のブルーライトを避け、6時間以上の睡眠を確保する 。


4. 
**温**: 38〜40℃のぬるめのお湯に浸かって体を温める 。



良い睡眠のためのヒント 

* 朝起きたら日光を浴びて体内時計をリセットする 。


* 夕方以降のカフェイン摂取（100mg以上）を控える 。


* 寝酒は睡眠の質を悪化させるため控える 。

## 減量中に意識してほしいこと（Page 12）

* 
**水分摂取**: 1日2リットルを目安に。脂肪燃焼には水分が必要です 。


* 
**ペース**: 急激な減量はリバウンドの元。1ヶ月に-1kgを目安にします 。


* 
**停滞期**: 体の防御本能による自然な現象です。あせらず取り組みを継続しましょう 。
